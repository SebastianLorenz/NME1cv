# Požadavky k zápočtu  
* maximálně 3 absence
* zápočtový úkol
  * zadání dostanete v průběhu dubna
  * odvzdání ideálně do konce června, maximálně do konce srpna, NE v září!
* je možné jít na zkoušku i před udělením zápočtu
